API Documentation

Endpoints

GET /api/logs - Retrieve backend log entries.

POST /api/logs - Submit a new log entry.

API Logs Page

The Error Logs button fetches backend error logs.

The Access Logs button retrieves access logs from the server.

Fail2Ban Page

The Jail Status button shows active jails.

The Banned IPs button lists blocked addresses.

The UML overlay on this page displays fail2ban-uml.svg.

GPU Support

Complex calculations may use the cuda-python library.

Manual Tests

Check each page loads correctly and the UML overlay appears when clicking the UML menu item. Verify links to Fail2Ban logs and GPU info work as expected.
