Развёртывание Django под Apache

Скрипт setup_django.sh автоматически настроит проект и Apache. Выполняется он по шагам:

  1. Создание директории проекта и установка прав.
  2. Создание виртуального окружения.
  3. Установка зависимостей Django и mod_wsgi.
  4. Инициализация проекта и приложения при их отсутствии.
  5. Обновление settings.py и добавление приложения.
  6. Выполнение миграций и сбор статики.
  7. Создание тестового представления и маршрутов.
  8. Генерация конфигурации Apache и перезапуск сервера.

Щёлкните по пункту, чтобы отметить его завершённым.

from js import document

def toggle_done(event):
    event.target.classList.toggle('done')

for li in document.querySelectorAll('#setup-steps li'):
    li.addEventListener('click', toggle_done)

def gen_script(event):
    proj = document.getElementById('proj').value or 'mysite'
    app = document.getElementById('app').value or 'app'
    host = document.getElementById('host').value or 'localhost'
    command = f'PROJECT_NAME={proj} APP_NAME={app} HOSTNAME={host} bash setup_django.sh'
    document.getElementById('output').textContent = command

document.getElementById('gen-btn').addEventListener('click', gen_script)
