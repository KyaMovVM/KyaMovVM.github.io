Что такое JSON Schema

JSON Schema описывает структуру JSON-документов и позволяет валидировать данные. Он используется для проверки входных данных, генерации документации и автокомплита в редакторах.

Пример

{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "type": "object",
  "properties": {
    "name": {"type": "string"},
    "age": {"type": "integer", "minimum": 0}
  },
  "required": ["name", "age"]
}
